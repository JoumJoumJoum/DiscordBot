# DiscordBot

A simple discord bot I made for the Fifa World Cup to bet among my friends on matches.

## Setup

### Requirements
- Python 3.8+
- discord.py 2.7.1+

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/JoumJoumJoum/DiscordBot.git
   cd DiscordBot
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file in the root directory with:
   ```
   TOKEN=your_discord_bot_token
   ```

4. Run the bot
   ```bash
   python bot.py
   ```

## Environment Variables

- **TOKEN**: Your Discord bot token from the [Developer Portal](https://discord.com/developers/applications)
